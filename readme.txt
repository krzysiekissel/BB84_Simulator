Symulator BB84 

Aplikacja ma na celu symulowanie dystrybucji klucza kwantowego z podsłuchem zgodnie z protokołem BB84

W aplikacji możliwe jest zmienianie następujących parametrów:
- liczba fotonów/qubitów
- błąd/szum kanału kwantowego
- błąd detektora
- modifikacja strategi podsłuchu


Aplikacja wymaga zainstalowanego .NET Core 3.1 Runtime (https://dotnet.microsoft.com/download/dotnet-core/current/runtime)
